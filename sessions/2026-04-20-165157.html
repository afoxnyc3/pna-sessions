Recommendation

Build a minimum-viable transcript-to-skill deepagents skill with this stack:

  1. yt-dlp -- audio extraction with caption fast-path (free, near-instant for YouTube)
  2. faster-whisper large-v3 local (under ~10h/week) or Deepgram Nova-3 (above that threshold)
  3. HEDGEhog classifier -- external pass before extraction, not in-prompt instruction (EMNLP 2024 result)
  4. LLM x MapReduce with structured JSON state (ACL 2025 pattern) -- not prose-summary reduce
  5. Pydantic ProcedureStep with research_b's schema fields (includes failure_mode)
  6. cefriel-style KG intermediate before final SKILL.md render -- makes reduce a graph merge
  7. Dedicated modality-shift LLM pass with 5-8 few-shot narration/procedure pairs
  8. skillcheck or skrills-validate wired into CI

Cheapest validation: one 45-minute conference talk with a published blog post companion. Run the pipeline. Diff generated SKILL.md against a hand-written version from the blog post.
